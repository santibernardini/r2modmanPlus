r2modmanPlus is a simple mod manager that helps you manage mods for several games. It works with popular mods from Thunderstore. With r2modmanPlus, you can easily add, remove, and update mods without messing with game files or settings.

## 🔧 Common Troubleshooting

- **The app won't open:** Make sure you extracted all files from the ZIP. Also, check that your .NET Framework is up to date.
- **Mods don’t appear in the game:** Confirm you selected the correct game folder in the profile settings. Some games require you to run through Steam or their launcher after installing mods.
- **Updates fail to download:** Check your internet connection and try again. Antivirus software might block the app; temporarily disable it if needed.
- **Game crashes with mods:** Disable all mods and add them back one by one to find the problematic one.
